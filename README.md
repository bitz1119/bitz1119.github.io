# Bittoo Aggarwal — Portfolio

ex-Geekster founding team, ex-Amazon SDE.

**Live →** https://bitz1119.github.io/

## What's inside

- `index.html` — the entire site, hand-built in HTML, CSS and a touch of vanilla JS.
- `assets/` — portrait, speaking photos, and compressed product-demo videos.

## Design

- **Type:** Fraunces (display) · Instrument Sans (body) · JetBrains Mono (labels)
- **Palette:** Warm parchment background · ink black · vermillion accent.
- **Aesthetic:** Editorial / print-magazine, asymmetric grid, generous typography.

## Run locally

No build step. Just open `index.html`, or serve the folder:

```bash
python3 -m http.server 8080
# then open http://localhost:8080
```

## Deploy

This repo is deployed via **GitHub Pages** from the `main` branch root.
